# 範例索引

本部分提供 Semantic Kernel Graph 套件中所有可用範例的完整概述。每個範例都示範了特定的功能和模式，您可以將其作為建立自己的圖形工作流程的參考。

## 快速入門

本文件中的範例提供了 Semantic Kernel Graph 套件功能的全面演示。每個範例包括：

* **完整的程式碼片段**，您可以複製和調整
* **分步說明**，說明程式碼如何運作
* **不同場景的設定範例**
* **最佳實踐**和使用模式

瀏覽下方按類別劃分的範例，為您的使用案例找到合適的起點。

## 執行範例

### 先決條件

1. **.NET 8.0** 或更新版本
2. **OpenAI API 金鑰**（在 `appsettings.json` 中設定）
3. **已安裝 Semantic Kernel Graph 套件**

### 設定

在範例專案中建立或更新 `appsettings.json`：

```json
{
  "OpenAI": {
    "ApiKey": "your-api-key-here",
    "Model": "gpt-3.5-turbo",
    "MaxTokens": 4000,
    "Temperature": 0.7
  }
}
```

### 範例執行流程

1. **設定**：使用圖形支援進行核心設定
2. **建立圖形**：使用節點和邊建立圖形結構
3. **執行**：使用輸入參數執行圖形
4. **監控**：觀察執行流程並收集指標
5. **結果**：處理和顯示執行結果

## 疑難排解

### 常見問題

* **API 金鑰遺失**：檢查 `appsettings.json` 設定
* **套件相依性**：確保安裝了所有必要的套件
* **記憶體問題**：大型圖形可能需要增加記憶體限制
* **逾時錯誤**：調整複雜工作流程的執行逾時
