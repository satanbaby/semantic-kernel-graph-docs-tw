## 實施檢查清單

對於每個範例，請檢查是否包含：

* [ ] **概念和技術**：所使用元件和模式的定義（附參考連結）
* [ ] **參考資源**：相關 API 和指南的連結
* [ ] **先決條件**：技術要求和所需知識
* [ ] **步驟**：包含主要程式碼片段的逐步實施
* [ ] **預期輸出**：範例的預期結果
* [ ] **疑難排解**：常見問題和解決方案
* [ ] **建議的變體**：進階模式（如適用）
* [ ] **跨連結章節**：包含參考和指南連結的「另請參閱」章節

## 實施注意事項

1. **一致性**：在所有範例中保持相同的格式和風格
2. **活躍連結**：所有內部連結必須有效並指向現有頁面
3. **可執行程式碼**：程式碼片段必須編譯並反映當前的 API
4. **導覽**：促進範例、指南和 API 參考之間的導覽
5. **搜尋**：包含相關術語以便於文件中的搜尋
