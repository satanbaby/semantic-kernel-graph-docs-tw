使用統一的正體中文技術術語翻譯：

- **Graph** → 圖
- **Node** → 節點

- **Workflow** → 工作流程
- **State** → 狀態
- **Execution** → 執行
- **Kernel** → 核心
- **Agent** → 代理

4. **可讀性**: 確保翻譯後的文件易於理解
